## Скрипты

### `yarn start`

Режим разработки.<br />
Открывается автоматически по адресу [http://localhost:3000](http://localhost:3000) .

Поддерживает HotReload.<br />
Ошибки линтинга выводятся в консоль.

### `yarn build`, `yarn test`, `yarn eject`
